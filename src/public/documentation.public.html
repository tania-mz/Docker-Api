Documentación de API

POST /login

Endpoint para iniciar sesión y obtener un token de acceso.

Devuelve en JSON:

Nombre Descripción
user_id uuid del usuario
token JWT que representa la sesion

Establece un JWT en la cookie 'access_token'.

Errores:

400 - Request malhecho o malformado

401 - Nombre o usuario incorrecto

POST /logout

Endpoint para cerrar sesión.

Borra la cookie de sesion.

POST /register

Endpoint para registrar un nuevo usuario.

Recibe en el body: { "username": "string", "email": "string", "password": "string" }

No devuelve ningún valor.

Errores:

400 - Request malhecho o malformado

401 - El usuario ya existe

GET /users

Obtiene todos los usuarios.

Respuesta Exitosa:

[
  {
    "user_id": "string",
    "username": "string",
    "email": "string"
  }
]
GET /users/:id

Obtiene los datos de un usuario específico.

Respuesta Exitosa:

{
  "user_id": "string",
  "username": "string",
  "email": "string"
}
GET /users/:id/sections

Obtiene todas las secciones de un usuario.

Respuesta Exitosa:

[
  {
    "section_id": "string",
    "section_name": "string"
  }
]
GET /users/:id/sections/:sectionId

Obtiene una sección específica de un usuario.

Respuesta Exitosa:

{
  "section_id": "string",
  "section_name": "string"
}
GET /users/:id/sections/:sectionId/tasks

Obtiene todas las tareas de una sección específica de un usuario.

Respuesta Exitosa:

[
  {
    "task_id": "string",
    "description": "string"
  }
]
GET /users/:id/sections/:sectionId/tasks/:taskId

Obtiene una tarea específica de una sección de un usuario.

Respuesta Exitosa:

{
  "task_id": "string",
  "description": "string"
}
POST /users/:id/sections

Crea una nueva sección para un usuario.

Cuerpo de la Solicitud:

{
  "sectionName": "string"
}

Respuesta Exitosa:

{
  "section_id": "string",
  "section_name": "string",
  "user_id": "string"
}
POST /users/:id/sections/:sectionId/tasks

Crea una nueva tarea para una sección específica de un usuario.

Cuerpo de la Solicitud:

{
  "taskName": "string"
}

Respuesta Exitosa:

{
  "task_id": "string",
  "task_name": "string",
  "section_id": "string"
}
